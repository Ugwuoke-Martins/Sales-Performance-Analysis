Sales Performance Analysis
Overview
This project analyses sales performance data from a global superstore using Excel and Power BI. The goal was to identify top performing regions, categories, customers, and segments.

Tools Used
Microsoft Excel
Microsoft Power BI

Dataset
Sample Superstore dataset from Kaggle
9,994 rows of retail sales data

Key Insights
West region generates the highest sales at $725K
Technology is the most profitable category
Sean Miller is the top customer by sales value
Consumer segment accounts for 50% of total sales

Dashboard Features
Interactive slicers for Region, Category, and Ship Mode
Cards showing Total Sales, Total Profit, and Total Quantity
Six visualisations covering region, category, customer, and segment analysis
